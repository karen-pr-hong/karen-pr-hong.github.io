Engagement Dashboard

An interactive dashboard that allow users to customize filters and build views and graphs of their own.
November 2019

APG Asset Management has been building its in-house knowledge management system for years, recording investment cases, engagement activities, and data feeds from third party vendors. The knowledge management system provide a wide range of features but lack a high level overview. I initiated the project to build an interative dashboard on Qlik Sense, a data visualization and business intelligence tool. The dashboard is connected to the SQL Server and will be able to pull out the latest available data in the database. Managers are able to monitor portfolio status on the colorful and user-friendly dashboard and then source back to the original knowledge management system to further dive into details, if needed.

KickStarter Database

A web application that interfaces with a relational database.
June 2016

<

For this project, I worked in a team with 3 other students to design the conceptual schema of a database, populate the database, then embed it in a web application. Our project successfully delivers a web application that provides useful features for pokemon trainers and administrators. Administrators, such as professors and gym trainers, can log in to access statistical data about the world and modify the database to reflect changes. They are allowed to update, add, and delete information as necessary. Regular trainers can view information about themselves, such as a list of their pokemon and items. Visitors can also look up various information on pokemon and pokemon species on the Pokemon page.

My contributions to the project:

  • Participated in the design discussions for the database
  • Populated the database
  • Structured several queries for the retrieval and addition of information
  • Implemented the functionality on the administration page
